# Image Caption Generator

An AI-powered Image Caption Generator developed using Flask and Transformers.

## Features
- Upload images
- Generate automatic captions
- Deep learning based image understanding
- Responsive UI

## Tech Stack
- Python
- Flask
- Transformers
- PIL
- HTML
- CSS

## Installation

Clone repository:

```bash
git clone https://github.com/patilsamruddhi17/image-caption-generator.git
```

Install dependencies:

```bash
pip install flask transformers pillow torch
```

Run:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
